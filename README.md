# 🤖 CommitCraft

> Generate beautiful, standard-compliant Git commit messages locally using AI in less than 70 lines of code. Zéro cloud, 100% privacy.

## ✨ Features
- 💻 **100% Local:** Powered by Ollama (Mistral/Llama3). No API keys required.
- ⚡ **Lightweight:** Pure Python, zero third-party dependencies.
- 🎯 **Standardized:** Strictly follows the Conventional Commits specification.
